Skills
Language
JavaScript

ES6이후 문법을 활용하여 코드를 작성하고, 웹 기능을 구현할수 있습니다.

이벤트 처리, 비동기 작업을 통해 사용자의 인터랙션에 반응하는 웹 애플리케이션을 개발할 수 있습니다.

TypeScript

기본적인 타입 선언 및 라이브러리에 declare를 사용하여 전역 변수의 타입을 정의하여 사용한 경험이 있습니다.

BackEnd
NestJS

TypeScript와 NestJs 프레임워크를 사용하여 서버 애플리케이션 개발 경험이 있습니다.

NestJS의 모듈 시스템, 데코레이터 등을 활용할 수 있고, NestJS의 다양한 내장 기능과 미들웨어를 이용하여 데이터 유효성 검사(DTO), 예외 처리(filter) 등을 구현할 수 있습니다.

Express

TypeScript 및 JavaScript를 이용하여 Express 라이브러리 기반의 서버 애플리케이션 개발경험이 있습니다.

FrontEnd
React

JavaScript 및 TypeScript를 이용하여 react기반의 웹 페이지를 구현할 수 있습니다.

Tailwind CSS

HTML
